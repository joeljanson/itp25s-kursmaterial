# Hej ITP25S!

I detta repository ligger alla filer och kodexempel som vi går igenom. Detta kan ni kopiera genom att använda Git (och göra det som kallas för att klona ett repository). Alternativt kan ni klicka på den gröna knappen uppe i högra hörnet där det står "Code" och välja "Download as zip".

Om ni laddar hem som zip behöver ni ladda hem hela repositoryt efter varje lektion för att få den senaste koden. Fördelen med git är att man kan uppdatera sin lokala kod med ändringar som skett i det som ligger på Github :)
