Tre olika sätt att använda CSS

1. Inline CSS

Detta är styling direkt i HTML-elementet med style-attributet.

Fördelar: Snabbt att testa, högst prioritet

Nackdelar: Svårt att underhålla, kan inte återanvändas

2. Intern CSS (med <style> tag)

Detta är CSS som skrivs i en <style> tag i <head>.

Fördelar: All CSS på samma ställe, enklare än inline

Nackdelar: Kan bara användas på en sida, större HTML-filer

3. Extern CSS (importerad fil)

Detta är CSS som ligger i en separat fil (t.ex. style.css) och importeras med <link> i <head>.

Fördelar: Återanvändbar på flera sidor, renare HTML, lättare att underhålla

Nackdelar: Ytterligare HTTP-request (men minimal påverkan)
