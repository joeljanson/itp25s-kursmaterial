1. Grundläggande Selektorer

Det finns tre huvudtyper av grundläggande selektorer i CSS.

Element-selektor

Element-selektorn används för att styla alla element av en viss typ. I CSS skriver du bara elementnamnet.

Exempel: Alla <p>-element på denna sida har fått en blå färg via element-selektorn.

Detta är ett vanligt paragraf-element som påverkas av element-selektorn.

Alla paragraf-element får samma styling automatiskt.

ID-selektor

ID-selektorn används för att styla ett specifikt element med ett unikt ID. I HTML använder du id="mitt-id" och i CSS skriver du #mitt-id.

Exempel: Detta paragraf-element har id="special-paragraph" och får därför en unik orange bakgrundsfärg.

Viktigt: Ett ID ska bara användas en gång per sida - det är unikt! Varför? Det kommer vi prata mer om när vi pratar om JavaScript.

Class-selektor

Class-selektorn används för att styla flera element som delar samma klass. I HTML använder du class="min-klass" och i CSS skriver du .min-klass.

Exempel: Detta paragraf har klassen highlighted.

Detta är ett annat paragraf-element med samma klass highlighted.

Flera element kan ha samma klass och får då samma styling.

Fördel: Du kan återanvända samma styling på flera element!

2. CSS Specifitet

När flera CSS-regler kan gälla för samma element, vinner den mest specifika regeln. Ordningen är: Element < Class < ID

Exempel på Specifitet

Detta paragraf-elementet har ingen klass eller ID, så det påverkas bara av element-selektorn.

Detta paragraf-elementet har klassen spec-example och får därför en grön bakgrund (class-selektor vinner över element-selektor).

Detta paragraf-elementet har både klassen spec-example OCH id:et spec-winner. ID-selektorn vinner eftersom den är mer specifik, så den får en lila bakgrund.

Regel: ID (#) > Class (.) > Element (p, h1, etc.)

Kombinera Selektorer

Du kan också kombinera selektorer för att bli mer specifik:

Detta paragraf-elementet är INNE i en div med klassen box.

Detta paragraf-elementet är UTANFÖR div:en och påverkas inte av kombinationen.

Förklaring: .box p betyder "alla p-element som är inne i ett element med klassen box".

3. Avancerade Selektorer

Pseudo-klasser låter dig styla element baserat på deras tillstånd eller position.

:hover - Muspekaren över elementet

När du håller muspekaren över ett element kan det ändra utseende.

Håll muspekaren över denna text för att se effekten! (Den blir större och får en bakgrundsfärg)

:first-child och :last-child

Dessa selektorer väljer första respektive sista barnet i en container.

  • Detta är ett vanligt listelement
  • Detta är ett vanligt listelement
  • Detta är sista listelementet (får också speciell styling)

:nth-child() - Välj specifikt barn

Med :nth-child() kan du välja ett specifikt element baserat på dess position.

  • Element 1
  • Element 2 (detta är det andra elementet - får speciell färg)
  • Element 3
  • Element 4
  • Element 5

Tips: :nth-child(2) väljer det andra elementet. Du kan också använda even (jämna) eller odd (udda).
